# UI/UX Pro Max Skill
The UI/UX Pro Max skill is installed in `.agents/skills/ui-ux-pro-max`.
When generating designs, dashboards, or components, you MUST use the python script:
`python3 .agents/skills/ui-ux-pro-max/scripts/search.py "<keywords>" --design-system`
to retrieve the comprehensive design system, color palettes, and UX guidelines BEFORE writing UI code.
